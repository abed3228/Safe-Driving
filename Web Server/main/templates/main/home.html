Welcome to Safe Driving

by Alnkib Abed

We are building a car alcohol testing system

Our goal is that we all drive in a safe environment

How the system works
01

Before starting the vehicle

Press the alcohol test button

02

The system will ask the user to exhale into the device

03

The system will analyze the level of alcohol

04

If the alcohol level is normal, the system will switch to entering a secret code for the vehicle

Otherwise it will move to the next stage

05

If the alcohol level is quite close to the allowable The system will ask the user to wait

Otherwise it will move to the next stage

06

The level of alcohol in the blood is very high The system will send a message to the vehicle owner (registered in the system)

That the driver is drunk and the location of the vehicle

During all this time the vehicle will not start
